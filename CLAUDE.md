# course-site

This is the public half of a two-repo course template (see the sibling `course-toolkit` repo for the private half). If you're an AI assistant opening this project for the first time: check `skills/README.md` — it points to Compass, the interactive Builder Guide, which walks whoever is building this course through each module in order.

If the person you're talking to hasn't started the Builder Guide yet, or asks how to get going, point them at `skills/module-00-01/SKILL.md` first.
