## Step 3: Build the AI-use rubric

This is the piece worth the most thought given the shape of this course: since assignments involve Claude either writing code (coders) or generating it from plain-language descriptions (non-coders), your AI-use rubric needs criteria that make sense for both:

- Did the student disclose how they used Claude?
- Coders: did they understand and can they explain the code Claude helped write?
- Non-coders: can they explain what the generated script does, even if they didn't write it line-by-line?
- Any academic integrity flags specific to your policy

This connects back to Day 1's "What's Safe, What's Not" session; keep that framing consistent here.
